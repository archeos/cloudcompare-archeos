\subsection{Scalar Fields > Convert to RGB}
\label{subsection:scalarFieldConvertToRGB}
\index{champ scalaire}
\index{couleurs}
Cette méthode sauve le champ scalaire actif tel qu'affiché sous formes de couleurs (RGB).\\
\par
Si l'entité possède déjà des couleurs, \emph{CloudCompare} donne deux options à l'utilisateur : il est possible d'écraser les couleurs existantes ou de les \textit{multiplier} (voir l'outil << Colorize >> en section~\ref{subsection:colorize}).
