\subsection{3D Views > Tile}
\label{subsection:viewsTile}

\par
Cette commande permet de partionner l'espace d'affichage entre les différents contextes graphiques\index{contexte graphique}
ouverts. Les contextes sont disposés de manière à ce que l'espace d'affichage soit entièrement rempli,
et qu'il n'y ait aucun chevauchement entre contextes (ils forment une mosaïque).
\\
\par
Note : Cette organisation est utile pour visualiser tous les contextes simultanément.
