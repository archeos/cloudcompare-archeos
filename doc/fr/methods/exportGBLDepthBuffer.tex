\subsection{Sensor > Ground-Based Lidar > Export depth buffer}
\label{subsection:exportGBLDepthBuffer}

\index{profondeur, carte de}
Cette fonction permet d'exporter la carte de profondeur associée à un \emph{capteur} sous la forme
d'un fichier texte.\\
\par
L'utilisateur est invité à spécifier un nom de fichier dans lequel seront sauvegardées toutes les
informations relatives à la carte de profondeur (voir section~\ref{subsection:depthMapFileDescription}).
