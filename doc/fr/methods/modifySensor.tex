\subsection{Sensor > Modify}
\label{subsection:modifySensor}

\index{capteur}
Permet de modifier les paramètres d'une entité \emph{capteur} (\textit{GBL sensor}).\\
\par
L'utilisateur peut mettre à jour les paramèters de l'entité \emph{capteur} via la même boîte de dialogue que dans le cas de la création d'une entité \emph{capteur} (voir~\ref{subsection:sensorProjection}).
