\index{couleurs}
L'utilisateur à le choix entre appliquer une rampe par défaut (figure~\ref{fig:heightRampExample})
ou alors une rampe dont il définit les deux couleurs extrêmes (figure~\ref{fig:heightRampDlg}).
Il faut pour cela désactiver la case à cocher \emph{Use default ramp}. Les deux couleurs extrêmes du dégradé peuvent alors être définies en cliquant sur les boutons colorés \emph{First color} et \emph{Second color} (qui
font apparaître des interfaces de sélection de couleur équivalentes à celle de la méthode \emph{Set Unique} - Cf. section~\ref{subsection:setUniqueColor}).\\
\par
Via la liste déroulante \textit{direction}, l'utilisateur doit aussi définir selon quelle direction le dégradé sera appliqué (parmi les 3 directions principales : X, Y ou Z).
